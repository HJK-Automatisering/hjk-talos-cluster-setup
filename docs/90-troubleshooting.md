# Fejlfinding og FAQ

## Almindelige problemer

- Netværksfejl: Tjek VLAN og firewall-regler.
- Storage: Kontroller diskene og Ceph OSD status.
- Adgang: Tjek brugertilladelser og kubeconfig.

## Tips

- Brug `talosctl logs` og `kubectl logs` til fejlsøgning.
- Dokumentér alle ændringer og løsninger løbende.
