# HJK Talos Cluster Setup

Velkommen! Denne dokumentation beskriver, hvordan du opsætter en bare-metal Kubernetes-node med Talos, Cilium, Rook Ceph, Argo CD og andre nødvendige teknologier.

Brug denne guide som opslagsværk og trin-for-trin vejledning til at gentage opsætningen i fremtiden.
